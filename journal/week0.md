   ###¤ 7 - Install Aws cli
   
   I Installed AWS CLI on Ubuntu 20.4 using  the following **command**
           
   ```
   curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
   unzip awscliv2.zip
   sudo ./aws/install 
        
   ```
